## :droplet: :bar_chart: Monitoramento do Nível do Resevatório de Água 

Este projeto é contado de nível de Água, que permite ao usuário coloque o status do nível de agua com base em nível 1, 2, 3, 4 e 5 e o programa mostra a mensagem com lembrete com cor, correspondente á cada nível.

## :bar_chart: diferencial do Programa:
O programa apresenta níveis de água de 1 á 5, caso o usuário apresente um valor diferente de nível ele mostrara uma mensagem de erro. 

## ▶️ Como executar o programa:

 -Coloque o nivel de agua do reservatório de 1 á 5. 

 -Caso coloque um nível diferente do solicitado o programa exibira uma menssagem de erro com á orientação correta. 
 
 -O programa exibira uma menssagem com cores correspondente á cada situação.
